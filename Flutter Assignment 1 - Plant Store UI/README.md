## Run locally

### Prerequisites
- Flutter (stable channel)
- Xcode with iOS Simulator (for iOS)

### Install dependencies
```bash
flutter pub get
```

### Run on iOS Simulator (iPhone 16 Pro)
```bash
flutter run -d "iPhone 16 Pro"
```
